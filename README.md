# linkedin-cv

A beautiful CV template that emulates the LinkedIn UI.

## Installation

```typ
#import "@preview/linkedin-cv:0.0.1": *
```

## Usage

```typ
#show: linkedin-cv.with(
  firstname: "Your",
  lastname: "Name",
  email: "hello@example.com",
  mobile: "01234 567890",
  github: "your-github-username",
  linkedin: "your-linkedin-username",
)
```
